# Portfólio – Lógica de Programação em Python

Este repositório reúne exercícios desenvolvidos em **Python** com foco em **lógica de programação**, como parte do módulo de **Inteligência Artificial Generativa** do curso de extensão em **Computação em Nuvem (AWS)**.

Os códigos têm como objetivo fortalecer o raciocínio lógico, base essencial para aplicações em **IA**, automação e sistemas inteligentes.

---

## Objetivos
- Desenvolver pensamento lógico e computacional  
- Praticar estruturas condicionais e laços de repetição  
- Criar funções e validações básicas  
- Consolidar fundamentos para aplicações em IA Generativa  

---

## Conteúdos Abordados
- Variáveis e tipos de dados  
- Entrada e saída de dados  
- `if / elif / else`  
- `for` e `while`  
- Funções (`def`)  
- Manipulação de strings e números  

---

## Tecnologias
- Python 3  
- Git & GitHub  
- VS Code / Terminal  
- Conceitos introdutórios de IA Generativa  
- Fundamentos de AWS
